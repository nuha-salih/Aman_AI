## 📌 The Problem
Financial scam calls, SMS phishing, and AI-generated deepfake voice fraud are rising sharply across Saudi Arabia and the wider Gulf region — yet almost no Arabic-language tools exist to detect them in real time. **Aman AI** ("أمان") closes that gap.

## 💡 The Solution
Aman AI is a **dual-sided AI platform**:
- **B2C (Consumer Protection):** Real-time analysis to flag scams before damage is done.
- **B2B (Enterprise Security):** Anonymized scam patterns to help banks train employees and customers against real fraud tactics.

## ✨ Key Features
- **Unified analysis:** Multi-modal support (SMS, URL, Audio).
- **AI-Driven Scoring:** Real-time risk assessment and confidence levels.
- **Privacy-First:** Data processed securely with local extraction for sensitive biometrics.
